__**Report von Quests**__
Das Kommando lautet:
```
!quest <Quest + Reward + Ort (desto genauer, desto besser)>
```
Beispiel:
```
!quest Fange 3 Myrapla/knofensa - Reward: Tangela - Pokestop: "Adler" (Technische Fakultät)
```

__**Report von Raids**__
Das Kommando lautet:
```
!raid <Boss + Ort + Hatch/Despawn Zeit>
```
Beispiel:
```
!raid Rayquaza - Jump & Twist (Technische Fakultät) - hatch 12:00
```

__**Report von Nestern**__
Das Kommando lautet:
```
!nest <Ort + Pokemon>
```
Beispiel:
```
!nest Escholzpark - Rihorn
```

__**Report von Rare und high IV pokemon**__
Das Kommando lautet:
```
!rare <Pokemon + IV + Ort>
```
Beispiel:
```
!rare Dratini 100% - Rathausplatz :D
```
